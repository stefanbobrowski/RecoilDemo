# Recoil Demo

Recoil demo app.

Package includes:

- react
- react-router-dom
- recoil
- redux
- redux-saga
- sass
- eslint
- prettier
- recoil-logger

## Getting Started

From the root folder...

```

npm install

npm start

```

Open `http://localhost:3000` in your favorite browser.
